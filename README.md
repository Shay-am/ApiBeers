## Installation

You need to install [node.js](https://nodejs.org/en/download/)

Clone the repository using a terminal

```bash
git clone https://github.com/Shay-am/ApiBeers.git
```
Install all dependencies

```bash
npm install
```

Run the project

```bash
npm start
```
